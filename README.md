# Reactron-Vite

React+Electron+Viteのボイラープレートです

### nodejsのインストール
```bash
 winget install CoreyButler.NVMforWindows
 nvm install 20.18.1
 nvm use 20.18.1
```


### パッケージインストーラーyarnのインストール
```bash
 npm install -g yarn
```

### モジュールのインストール


```bash
 yarn install
```

### アプリの起動方法

```bash
 yarn dev
```

### ビルド(インストーラーの作成)

```bash
 # windows
 yarn build:win

```
